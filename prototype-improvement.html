Prototype Improvement

Prototype Improvement

For the feedback and suggestion from the participants, we use them for improve our prototype including

  • Users found the "Password" label unclear, suggesting renaming it to "Change Password".
  • Suggested adding two clear options: one for changing the password with current and new inputs, and another for resetting via email for forgotten passwords.
  • Highlighted confusion due to grouped password fields in the profile settings.
  • Recommended making the password process more intuitive and separated to reduce user hesitation.
