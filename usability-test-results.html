Feedback and Suggestion from Participants

  • Change password process is confusing.
  • Suggestion: Create a separate field or make the label clearer for the change of password section.

Summarize Usability Test

During the usability test, we observed varying levels of user clarity across tasks. In the first task, the sign-up process, some users experienced initial confusion but were ultimately able to complete the task successfully. In the second task, scanning medicine, users navigated the interface with ease, as the scan button was clearly visible at the center of the bottom navigation, allowing them to complete the task without difficulty. However, in the third task, related to password management, users hesitated to interact with the password input field. This confusion arose because the field was only labeled "Password," which led to uncertainty about whether this was the correct place to change their password. A clearer label or a separate field would alleviate this confusion.
